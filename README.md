# Library Book Management System (Java)

This project is a simple console-based Java program that helps manage basic library operations such as adding books, searching for books, issuing them, returning them, and viewing the current list. The purpose is to organize book records in a structured way without using manual registers.

## Features
- Add new books
- Search by title or ID
- Display available books
- Issue a book
- Return a book
- Basic input validation

## How to Run
1. Open the project folder.
2. Compile using:
   javac src/*.java
3. Run using:
   java src.Main

## Project Structure
- Book.java – stores book details  
- Library.java – contains operations like add, search, issue, return  
- Main.java – handles menu and user input  
- /screenshots – output screenshots  
- /recordings – optional demo video
